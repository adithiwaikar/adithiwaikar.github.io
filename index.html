Driver Drowsiness Detector

Our proposal involved neural signal analysis to detect driver drowsiness using Electroencephalogram [EEG]. EEG is a device used to read brain signals to observe the behavior of an individual. To achieve this, we interfaced EEG with micro controller to tap brain waves. Further analysis of these signals were used to detect drowsiness of the individual and the same was indicated using alerts in the form of alarm, vibration and the glowing of LED light.

The purpose of the driver drowsiness detection system is to alert the driver and prevent accidental hazards. This reduces the accidents caused due to fatigue driving. The above project received a grant of INR 1,00,000 from the Govt. of India.

Mentored by Prof. P. Usha, Ph.D.

Project Duration: Aug - Feb 2015


Heavenet - Smart Helmet

Automation and Testng of Primary Actuator Card

This project was carried out as a part of the B.E. Final Year Project at Bharat Electronics Limited, India (BEL). The project focused on automating the process of testing the Primary Actuator Card. This was achieved by designing LabVIEW programs to run continuously, controlling the various components to simulate the Signal Conditioning Cards. A database was created such that the software could access the required information as per the testing protocol and automate the entire testing process. Relay pins were assigned to facilitate the designing of a cable so that probe changes after every test could be avoided and the tests can be run continuously.

Duration: Jan - May 2016


Cricket Match Score Prediction

The aim of the project was to predict match scores between two Indian Premier League (IPL) teams based on previous IPL match statistics. Player vs player statistics and individual player statistics were scraped from ESPN CricInfo. The players were divided into clusters using Apache Mahout's K-means clustering algorithm. Using Apache Spark, a map reduce algorithm was written to calculate probabilities of occurrence of different events like 1s scored, 4s scored, wicket etc. Finally, given 2 IPL team rosters, a ball by ball simulation was done and a prediction algorithm was used to determine the final score for both the teams and declare the winner. A random process was initiated to select the events and determine the score based on the probabilities calculated previously. In this project, I worked on the Mahout clustering module and the Spark map reduce algorithm for computing event probabilities.
